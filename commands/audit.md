# Marketing Audit Command

Audit this website: $ARGUMENTS

Run these 5 agents in parallel:

1. **Content & Messaging** - Is the value proposition clear? Score it.
2. **SEO & Discoverability** - Title tags, meta descriptions, headings.
3. **Conversion Optimization** - CTAs, forms, friction points.
4. **Competitive Positioning** - Search 2-3 competitors and compare.
5. **Brand & Trust** - Does it look credible? Social proof?

Score each out of 100.
Print results like:
✓ Content & Messaging     — Score: XX/100
✓ SEO                     — Score: XX/100

Overall Score: XX/100
Save full report as MARKETING-AUDIT.md
