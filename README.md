# 🚀 AI Marketing Suite for Claude Code

Audit any website's marketing with one command.
Runs parallel AI agents and gives you a scored report instantly.

---

## ⚡ Install (One Command)
```bash
curl -sSL https://raw.githubusercontent.com/tariqhamad/ai-marketing-claude-code/main/install.sh | bash
```

---

## 🎯 How to Use

Open Claude Code and type:
```
/audit https://anywebsite.com
```

Claude launches 5 parallel agents and returns:

- ✓ Content & Messaging score
- ✓ SEO & Discoverability score
- ✓ Conversion Optimization score
- ✓ Competitive Positioning score
- ✓ Brand & Trust score

Full report saved as **MARKETING-AUDIT.md**
